Paragraphs should have running text with consistent spacing between lines as well as paragraphs across report. Recommended spacings are 1.5 points between lines, 12 points between headings-paragraphs, and 6 points between paragraphs. Any additional gaps created by inserting new lines (using Enter key) or spaces (using Space Bar key) are allowed only at the end of a page to ensure headings never start at the end of a page. Paragraphs should always be justify-aligned. All text in the report should be written in Times New Roman font with a recommended font size of 12.

Below are some best practices when using this report template -
Try to insert your draft report content within this template instead of trying to copy the styles from this template. This will reduce the formatting errors significantly.
It is preferable to use this template at the time of drafting your report content to reduce the additional step of formatting the report content once the draft is finalized.
This template is not designed in line with the MS Word Table of Contents feature. Please ensure to update the default styles when using the automated TOC.
Any number of sections, chapters or references can be included in this template. When adding an item, it is recommended to copy an existing item and edit it for ease of use.
Do Not Change the margins or layout of the page in the document. These margins are designed with respect to the binding specifications of the University.
When it is required to include a wide figure, rotate the figure and place it on a dedicated full page of its own (with captions, of course). Please try to avoid changing page layout for such figure pages when you are not accustomed with the layout settings..
